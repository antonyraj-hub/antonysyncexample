Claude Code in an Hour: A Developer's Intro
Claude Code lets you delegate real coding work directly from your terminal — exploring codebases, writing and running tests, fixing bugs, and committing code, all while keeping you in control at every step.

This hands-on session is built for developers who are comfortable in a terminal and have some coding experience, but are new to Claude Code. No AI expertise required — just bring a codebase you care about.

We'll walk through how Claude Code actually works under the hood, see it tackle a real development task end-to-end in a live demo, and leave with everything you need to start using
